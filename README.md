# Laboratorio #1: Introducción al Lenguaje de Programación PHP 🐘

Este repositorio contiene la resolución de los problemas prácticos asignados en el Laboratorio #1 para el curso de **Desarrollo de Software VII**.

---

## 🎯 Objetivos del Laboratorio
1. Verificar la instalación y configuración correcta de PHP mediante `phpinfo()`.
2. Identificar y aplicar la terminología básica de programación en el lenguaje PHP.
3. Resolver problemas prácticos aplicando la lógica de Entrada, Proceso y Salida en PHP.

---

##  Problemas Resueltos

###  Problema 1: Cálculo de Círculo
Script para solicitar/definir el radio de una circunferencia y calcular de forma automática tanto su área como su perímetro.
* **Fórmulas utilizadas:** $Area = \pi r^2$ y $Perímetro = 2\pi r$.

###  Problema 2: Conversor de Pulgadas a Centímetros
Aplicación que recibe una medida en pulgadas y realiza la conversión matemática al sistema métrico (centímetros).
* **Equivalencia:** $1 \text{ pulgada} = 2.54 \text{ cm}$.

###  Problema 3: Diagnóstico del Servidor
Comprobación de que el servidor web local (XAMPP / Wamp64) interpreta y ejecuta correctamente scripts de PHP usando la función `phpinfo()`.

###  Problema 4: Página Web Dinámica
Generación de una estructura básica de página web donde el contenido del cuerpo HTML se imprime de forma dinámica mapeando variables de PHP.

###  Problema 5: Procesamiento de Formulario HTML
Diseño de un formulario para captura de Nombre y Edad. Incluye un script receptor (`pagina2.php`) enfocado en:
* **Trimming:** Limpieza de espacios vacíos.
* **Lógica condicional:** Verificación de mayoría de edad para validar la facultad de votar en elecciones futuras.

###  Problema 6: Calculadora Básica (UI/UX)
Creación de una calculadora funcional con un diseño visual interactivo procesando operaciones aritméticas fundamentales como Suma, Resta y Multiplicación.
